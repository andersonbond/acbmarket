# Pilimarket Frontend

Ionic 8 + React frontend for Pilimarket.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm start
```

The app will be available at `http://localhost:8100`

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
